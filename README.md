# Programming Language Survey Form Site

### Code review solo project, 3/27/20, JavaScript functionality test

## by Ethan Firpo

## Description

This website is a form survey to determine which programming language would be best suited to the user. Designed to test functionality of JavaScript code basics.

Clone this repository.

Open index.html in the web browser of your choice.
